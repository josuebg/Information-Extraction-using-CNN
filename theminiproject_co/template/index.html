About The Project

Information Extraction has been one of the important task in Natural Language Processing (NLP). The said task have been accomplished by using sophisticated machine learning algorithms like Naive Bayes. We approached the problem using Convolutional Neural Networks (CNN) with least possible hyper parameter tuning on pre trained word vectors. We reached our goal with considerable results using CNN that is computationally much more efficient than the traditional approaches.
